  • Here are some of my projects in machine learning. These projects are my main research work this year.

    Neural Architecture Search (NAS)

    * A Device Generic Latency Predictor (DGL) is proposed, which can perform latency prediction for new devices without latency measurement.

    * An APP was developed with Android Studio to measure the inference latency of the network on mobile devices.

    * On the NAS-Bench-201 search space, the latency predicted by DGL and the actual latency achieve an R of 0.976, compared to 0.798 for the GCN predictor. On Google Pix3A, the architecture searched by DGL+OFA reached 78.3% top-1 on ImageNet 2012, which is 2.8% higher than OFA.

    * The output of this project, "DGL: Device Generic Latency model for Neural Architecture Search on Mobile Devices", has been submitted to the IEEE TMC and is undergoing Minor Revision.


    Dynamic DNNs

    * A set of equations is proposed, which can be solved quickly to obtain the highest accuracy architecture in the search space satisfying the latency constraint.

    * A dynamic network framework is proposed, which realizes real-time search and operation of effective architectures according to the device environment.

    * On the Nvidia Jetson TX2 GPU, the framework searched by the proposed framework achieves a top1 accuracy rate of 79.3% on ImageNet2012, which is 3.9% higher than that of AutoSlim. Meanwhile, the proposed method reduces the power consumption of network switching by about 60% compared to DydamicOFA.

    * The output of this project, "Neural Architecture Solver Based on Block-Architecture Accuracy Loss Model", has been submitted to the ICML 2023.

  • Here are some projects I have done on NLP.

    Lyric Prediction

    * 1000 lyrics (from Jay Chou's songs) are used as training samples. Character indices are built and onehot vectors are used as input to the network.

    * A recurrent neural network is trained and used for lyrics prediction. Perplexity (1 is best) is used as the evaluation index. After training for 150 epochs, the perplexity of the model reaches 1.06.

    * Further, a gated recurrent network is trained and used for lyrics prediction. The perplexity of the model reaches 1.02 after 120 epochs of training.

    * This project is my work when I participated in the 2020 Public Welfare AI Learning Together Training Camp.


    Machine Translation

    * The SuperTransformer search space proposed by HAT is used to search for a suitable architecture.

    * The accuracy predictor of HAT and the latency predictor of HELP are used for searching.

    * A BLEU score of 27.2 is achieved on WMT14-EN-DE, and the target device is NVIDIA Titan RTX.

    * This project is part of my work on Neural Architecture Search.

  • Here are some interesting projects I've done on computer vision. These projects are all personal attempts.

    Image Segmentation Based on Level Set

    * Use level set segmentation algorithm to complete image segmentation with python. Excellent segmentation is achieved on multiple classic images such as cups, twocells and vessels.


    Image Retrieval Based on Inverted Index

    * The SIFT features of 1000 images are randomly sampled. And the hierarchical k-means algorithm is used to train the visual codebook (codebook size is 10000).

    * L2 normalization is used to process the visual histogram.

    * The inverted index is used for the final output.


    Handwritten Word Recognition

    * A multinomial logistic regression model is used for image classification to recognize each letter separately.

    * The conditional probability formula for words is constructed by considering the letter classification and the probability of consecutive occurrence of two adjacent letters.

    * The Viterbi algorithm is used to find the sequence of letters that maximizes the conditional probability of a word to achieve word recognition.


    Contour Extraction

    * Laplacian and Sobel operators are used to extract contours.

    * The program is optimized to be more suitable for GPU parallel computing, and obtains a speedup ratio of 30 compared to CPU.

  • Here are some of my projects in mathematical modeling.

    Invasive Insect Image Recognition

    * Leading 3 team members to propose a TRML (Transfer-Residual-network-Maximum entropy-Learning) model that can effectively identify invasive insects in a large amount of information.

    * A formulating comprehensive scoring mechanism is proposed, which considers the ResNet image classification model based on transfer learning and the MaxEnt maximum entropy model based on environmental similarity.

    * On the data of Asian giant hornet invasion in North America, the accuracy rate of the proposed model reaches 94.25%, which is 5.74% and 10.75% higher than using image recognition and environment recognition, respectively.

    * The project won the Honorable Mention in the 2020 MCM/ICM.


    Transportation Planning

    * Lead two team members to transform the transportation problem with transshipment center into the minimum spanning tree model with the first branch fixed, and propose an improved Kruskal algorithm to solve it.

    * On the transportation data sample of Mengniu Company, the proposed method saves about 84.23% of the cost compared with the transportation scheme obtained by the distance judgment method.

    * This project won the Second Prize in 2020 Asia and Pacific Mathematical Contest In Modeling.
